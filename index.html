GPU optimization with JPEG

This project introduces an efficient image processing pipeline designed for edge devices, improving speed by 2.8× with only a 2% loss in accuracy. It uses JPEG's DCT coefficients as input to a CNN and applies a Teacher-Student SSL approach to maintain performance while reducing computation.

Open-source ASR & TTS models for Azerbaijani

Pichilti‑base‑v1 offers automatic speech recognition using a self‑supervised fine‑tuned Whisper decoder, achieving lower character error rates and efficient performance on Azerbaijani audio. Meanwhile, SARA_TTS complements this by providing high-quality Azerbaijani text-to-speech, trained on BHOSAI’s own voice dataset (e.g., “TTS_book_sounds”), enabling natural voice synthesis for applications like audiobooks or voice assistants.

Unsupervised Domain Adaptation for Mars Rover Mission

This project tackles vision-based navigation for Mars exploration through two tasks: multi-task learning for rovers and semantic segmentation for drones, both enhanced by unsupervised domain adaptation. The rover model combines semantic segmentation and image classification using synthetic and real datasets (AI4Mars, PanCam), while the drone model uses synthetic training and real images from NASA’s Ingenuity logs.
